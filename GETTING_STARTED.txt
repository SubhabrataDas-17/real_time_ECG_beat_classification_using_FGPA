## ARCHITECTURE IMPLEMENTED

  ECG Stream (INT8, 180 samples @ clock rate)
       ↓
  [Shift Register: 5-tap buffer]
       ↓
  [Conv1D: 8 filters, kernel=5, INT8 weights, INT32 accumulator]
       ↓
  [ReLU: max(0, x)]
       ↓
  [MaxPool: pool size = 2, downsamples 180→90]
       ↓
  [Conv1D: 16 filters, kernel=5]
       ↓
  [ReLU]
       ↓
  [MaxPool: pool size = 2, downsamples 90→45]
       ↓
  [Global Average Pooling: 45 timesteps × 16 channels → 16-element vector]
       ↓
  [Dense Classifier: 16 inputs → 1 output + sigmoid]
       ↓
  Classification Result (INT16, range 0-127, representing probability)


## KEY FEATURES

  ✓ Batch Normalization Folding
    - BN parameters (γ, β, μ, σ) fused offline into Conv weights
    - Zero hardware cost on FPGA
    - Maintains mathematical equivalence
    - Detailed theory documented

  ✓ Quantisation (FP32 → INT8)
    - Per-tensor symmetric quantisation
    - Scale factors: S = absmax / 127
    - Weights clipped to [−127, 127]
    - < 1% inference error (validated)

  ✓ Streaming Datapath
    - One ECG sample per clock cycle
    - 5-tap shift register for efficient windowing
    - Pipelined pooling and averaging
    - ~350 cycle latency per 180-sample beat

  ✓ Fixed-Point Arithmetic
    - Inputs: INT8 (ECG samples, previous layer outputs)
    - Weights: INT8 (quantised)
    - Accumulators: INT32 (prevents overflow)
    - Outputs: INT16 (feature/probability values)

  ✓ Modular & Testable Design
    - Each layer in separate module
    - Unit testbenches for all modules
    - Integration testbench for complete pipeline
    - Standard Verilog (synthesizable)
